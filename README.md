# 📊 Customer Churn Prediction using Machine Learning

## 📌 Project Overview

This project focuses on predicting customer churn using machine learning techniques. The goal is to identify customers who are likely to leave a telecom service based on their demographic and service usage data.

---

## 🎯 Objectives

* Analyze customer data to identify key churn factors
* Build a predictive model using machine learning
* Evaluate model performance using standard metrics
* Provide insights to reduce customer churn

---

## 🗂 Dataset

* Dataset used: Telco Customer Churn Dataset
* Contains customer details such as:

  * Demographics (gender, senior citizen, etc.)
  * Account information (tenure, contract type)
  * Services subscribed (internet, phone, etc.)
  * Billing details

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* Google Colab

---

## 🔧 Project Workflow

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Encoding
4. Train-Test Split
5. Model Building (Random Forest Classifier)
6. Model Evaluation (Accuracy, Confusion Matrix, Classification Report)
7. Feature Importance Analysis

---

## 🤖 Model Used

* Random Forest Classifier
* Compared with Logistic Regression and Decision Tree

---

## 📈 Results

* Achieved high prediction accuracy
* Identified key factors affecting churn such as:

  * Contract type
  * Monthly charges
  * Tenure

---

## 📊 Key Insights

* Customers with month-to-month contracts are more likely to churn
* Higher monthly charges increase churn probability
* Long-term customers are less likely to leave

---

## 🚀 Future Improvements

* Hyperparameter tuning for better accuracy
* Deployment using Flask or Streamlit
* Real-time prediction system

---

## 📌 Conclusion

This project demonstrates how machine learning can be effectively used to predict customer churn and help businesses take proactive steps to retain customers.
